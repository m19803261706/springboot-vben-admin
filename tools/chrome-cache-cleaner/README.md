# Cache Cleaner - Chrome 扩展

一键清除页面缓存并强制刷新，方便前端开发测试。

## 功能

- 🔥 **清除全部并刷新** - 清除 localStorage、sessionStorage、Service Worker 缓存、浏览器缓存，然后强制刷新
- 🗑️ **仅清除 Storage** - 清除 localStorage、sessionStorage、IndexedDB
- 📦 **仅清除浏览器缓存** - 清除浏览器的 HTTP 缓存
- 🔄 **强制刷新** - 忽略缓存强制刷新页面

## 安装方法

1. 打开 Chrome 浏览器，访问 `chrome://extensions/`
2. 开启右上角的 **"开发者模式"**
3. 点击 **"加载已解压的扩展程序"**
4. 选择此文件夹 `tools/chrome-cache-cleaner`
5. 扩展图标会出现在浏览器工具栏

## 使用方法

1. 点击浏览器工具栏中的扩展图标
2. 选择需要的清除选项
3. 页面会自动刷新

## 快捷键（可选）

在 `chrome://extensions/shortcuts` 中可以为扩展设置快捷键。
